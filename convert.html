使用的配置模板说明：

使用 LIAN 配置模板，包含以下分组：

  • 👽 AI1, AI2 - AI相关网站分组
  • 📘 GitHub - GitHub相关分组
  • 👯‍♂️ TikTok - TikTok分组
  • 🙋 Telegram - Telegram分组
  • 📀 流媒体 - 包含YouTube、HBO、Disney、Netflix、Spotify等
  • 🌍 国外 - 国外网站分组
  • ➡️ 国内 - 国内网站分组

节点分组：

  • 所有-自动：全部节点自动测速
  • 港台日新韩-自动：香港/台湾/日本/新加坡/韩国节点自动测速
  • 台日新韩-自动：台湾/日本/新加坡/韩国节点自动测速
  • 区域自动：香港/台湾/日本/新加坡/韩国/美国等分区节点自动测速

※ 所有自动测速组都会每60秒测试一次，延迟阈值150ms
